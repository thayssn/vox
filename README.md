# voxustest

## Requirements


I recommend using `node@^10.15.1`, `npm@6.4.1` and `gulp-cli@2.0.1`.


## Running the project

```
	cd voxustes
	npm i
```

### Dev

```
	npm run dev
```
or simply

```
	gulp
```

### Prod

```
	npm run prod
```
or simply
```
	gulp production
```
